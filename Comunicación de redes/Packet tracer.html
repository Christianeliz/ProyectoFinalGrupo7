Luego de esto seleccionamos en la pestaña escritorio, y pulsamos Ip configuración:

En esta parte es donde se asigna el IP estáticas para que se puedan conectar nuestros dispositivos de manera correcta:

Una vez realizado esto cerramos la ventana y procedemos a realizar esto en todas nuestras Pc, para ello utilizaremos para mi ejemplo la IP: 192.168.1.5 la cual solo variará el ultimo numero es decir 192.168.1.7 Luego de eso para comprobar si está bien conectado y se está trasmitiendo realizamos un pin entre maquinas, para ello hacemos un clip sobre el dispositivo o Pc la cual desplegara una pantalla como la vez anterior y esta vez pulsaremos el botón Terminal y realizaremos un ping como muestra la imagen. Si sale como la muestra la imagen la conexión esta correcta.
